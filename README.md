# Kasa – Frontend (React + Vite) & API

📝 Description
Développement d'une application de location immobilière avec React. Ce projet illustre la structuration d'un projet front-end complet, du routage à la gestion des composants.

🎯 Objectif
Structurer un projet front-end complet avec React, en appliquant les bonnes pratiques de composants réutilisables et de routage.

🛠️ Travail réalisé
Création de composants réutilisables (cards, collapse, carrousel…)

Mise en place du routage avec React Router et des appels API

Ajout d'animations CSS pour améliorer l'expérience utilisateur

## 🚀 Lancer le frontend (React + Vite)

### Prérequis

- Node.js ≥ 18
- npm (fourni avec Node)

### Installation & démarrage

```bash
# à la racine du projet (dossier contenant package.json du frontend)
npm install
npm run dev
```
